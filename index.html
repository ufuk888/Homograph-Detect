How HomographDetect Works

A simple, non-technical guide to how we uncover hidden phishing links.

🎯 The Threat: Homograph Spoofing Attacks

Hackers often register fake websites using letters from foreign alphabets that look exactly like normal English letters. This tricks you into thinking you are on a safe, legitimate site.

a Latin (Normal)
looks exactly like
а Cyrillic (Fake)
✏️

1. You Paste a Link

Enter any suspicious domain, email address, or keyword into the search box. The tool safely reads the text without actually visiting the dangerous website.

🩻

2. We X-Ray Every Letter

Our engine breaks the text down character by character. It reveals the true DNA (the Unicode) of every letter to see exactly which alphabet it came from.

🚨

3. Spotting the Impostors

If a word is mostly English but contains a hidden foreign letter trying to blend in, the system catches the mismatch and instantly tags the fake letter as a THREAT.

🕵️

4. Catching Brand Fakes & Typos

Beyond mixed alphabets, the tool acts like a detective to catch common tricks: like swapping an "l" for a "1" (gmai1.com) or slightly misspelling famous brands.

📋

5. Your Final Security Report

You receive a clear, color-coded breakdown. We highlight exactly which letters are safe (Green) and which ones are dangerous traps (Red).

All analysis is executed entirely within your browser using JavaScript. Zero data is transmitted — this is a pure client-side application with no backend.

🛡️

Detection Scope & Limits

HomographDetect is a specialized tool, not a universal anti-phishing solution. Here is our exact scope.

✅ High-Confidence Detection
  • Unicode Script Anomalies If a domain uses characters from an unexpected alphabet, our engine catches it. By analyzing raw Unicode code points, we expose script-spoofing that is completely indistinguishable to the naked eye.
  • Punycode Unmasking We ensure that encoded Internationalized Domain Names (starting with xn--) are fully decoded and their true characters are exposed and analyzed.
⚡ Best-Effort (Bonus Features)
  • General Phishing & Typosquatting We are not a general phishing detector. While our engine tries to catch common typos (like gmai1.com) using Levenshtein distance and regex heuristics, these are just bonus features. We do not claim to catch every misspelled brand or novel evasion tactic.
